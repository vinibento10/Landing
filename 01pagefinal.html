Perguntas frequentes

Como funciona a integração com meu WhatsApp?

Integrável via API (Z-API, Meta, outros). Fazemos a configuração e testes para seu time usar.

O que é necessário para começar?

Conta no Tomik (se optar), número/conta de WhatsApp e permissões de calendar. Em 48–72h já conseguimos versão mínima funcionando.

Posso usar meus dados já existentes?

Sim — migramos ou integramos com Supabase/Tomik. Também temos templates prontos para começar do zero.
